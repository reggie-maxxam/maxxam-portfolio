## The Framework

All systems are built using the **WAT Architecture**:

- **Workflows** — Markdown SOPs defining what to do and how (stored in `workflows/`)
- **Agents** — AI reasoning layer that reads workflows and orchestrates tools
- **Tools** — Python scripts and n8n workflows that execute deterministically

More detail: [docs/wat-framework.md](docs/wat-framework.md)

## Tech Stack

The core stack across all client projects:

- **Automation:** n8n (self-hosted)
- **Voice AI:** RetellAI, Vapi
- **Scheduling:** Cal.com
- **Messaging:** Twilio (SMS), Gmail API
- **Data:** Google Sheets, Airtable
- **AI Models:** Claude (Anthropic), GPT-4o
- **Hosting:** IONOS, GitHub Pages, Hostinger
